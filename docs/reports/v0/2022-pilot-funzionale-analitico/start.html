Ops... quanta fretta di leggere il tuo report!

Di seguito sono fornite istruzioni dettagliate su come estrarre il report dal file ZIP:

  1. Trova il file ZIP: Inizia individuando il file ZIP sul tuo computer. Questo file avrà un'estensione ".zip".
  2. Fai clic destro sul file ZIP: Posiziona il cursore sopra il file ZIP e fai clic destro su di esso. Si aprirà un menu contestuale.
  3. Seleziona "Estrai" o "Estrai tutto": Nel menu contestuale, troverai un'opzione come "Estrai" o "Estrai tutto". Fai clic su questa opzione per avviare il processo di estrazione.
  4. Scegli una posizione di estrazione: Si aprirà una finestra che ti chiederà dove desideri estrarre i contenuti del file ZIP. Puoi scegliere una cartella esistente o crearne una nuova. Seleziona la cartella di destinazione facendo clic su di essa.
  5. Conferma e avvia l'estrazione: Una volta selezionata la cartella di destinazione, fai clic sul pulsante "Estrai" o "OK". Il processo di estrazione inizierà, e i contenuti del file ZIP saranno estratti nella posizione scelta.
  6. Trova i contenuti del file ZIP estratti: Dopo che l'estrazione è completata, puoi navigare nella cartella di destinazione che hai selezionato al passaggio 4. All'interno di quella cartella, dovresti trovare una cartella denominata html e il file start.html che desideri aprire.
  7. Apri il report start.html: Infine, fai doppio clic sul file start.html estratto per aprirlo nel tuo browser web.
